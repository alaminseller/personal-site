# Alamin Rafi - Digital Hub

Professional personal website for Alamin Rafi, Digital Marketing & Content Specialist.

## Overview

A clean, corporate-style personal website built with modern web technologies. Focuses on strong typography, editorial layout, and professional presentation of experience and skills.

## Tech Stack

- **Framework**: React + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel/Netlify ready

## Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/alaminseller/rafi-s-digital-hub.git
   cd rafi-s-digital-hub
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

## Project Structure

- `/src/components/professional`: Core UI components for the professional layout
- `/src/pages`: Page layouts
- `/public`: Static assets (Resume, Images)
